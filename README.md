# 📊 HR Analytics Dashboard (Power BI)

## 🔍 Project Overview
This project focuses on analyzing employee data to identify key factors affecting attrition, workforce distribution, and salary trends. The dashboard provides interactive insights to help HR teams make data-driven decisions.

---

## 🚀 Key Features
- Employee count and attrition overview  
- Attrition rate analysis (16.2%)  
- Department-wise filtering (HR, R&D, Sales)  
- Age group attrition analysis  
- Salary slab-based attrition insights  
- Job role-wise attrition distribution  

---

## 📊 Key Insights
- Highest attrition observed in the **26–35 age group**  
- Employees earning **below 5K salary** show maximum attrition  
- **Laboratory Technicians and Sales Executives** have the highest attrition  
- Attrition decreases as **years at company increase**
